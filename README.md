# 📦 Sistema de Gestión de Tienda

Aplicación web para gestionar información básica de un negocio, enfocada en simplicidad, rapidez de uso y una estructura clara para futuras mejoras.

---

## 🎥 Demo

👉 [Ver video de demostración](https://youtu.be/2TB3V0x6M08)

---

## 🎯 ¿Qué permite hacer?

- Agregar Notas
- Registrar proveedores
- Gestionar productos
- Administrar personal y turnos/horarios
- Realizar operaciones básicas: crear, editar y eliminar registros

El sistema está pensado para ser **intuitivo y funcional desde el primer uso**.

---

## ⚙️ Tecnologías utilizadas

- Backend: Django + Django REST Framework
- Frontend: Vue 3
- Contenedores: Docker + Docker Compose

---

## 🚀 Cómo levantar el proyecto

### Requisitos

- Tener instalado Docker
- Docker Compose

### Pasos

git clone https://github.com/Camilo-Martins/tu-tienda-v2
cd tu-tienda-v2
docker-compose up --build

### Acceder al sistena

http://localhost:5173/
